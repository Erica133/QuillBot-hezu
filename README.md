## 四、如何使用 QuillBot 做论文润色？

以下是我常用的论文润色步骤流程：

1. 写完英文草稿后，**将每段拆出来**。
2. 打开 QuillBot，进入 **Paraphraser 模块**。
3. 粘贴内容，选择「**Academic 模式**」。
4. 调整“同义词滑块”力度，点击「**Paraphrase**」。
5. **比对前后句子**，确认术语是否被误改（例如 CNN、LSTM）。
6. 复制结果粘贴回原文档。
7. 最后使用 **Grammarly** 做整体语法检查。
